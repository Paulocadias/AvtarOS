# Project Plan

## Vision
AvatarOS captures user biometrics to create a "Persona" database. Users can then generate personalized AI photos/videos of themselves using prompts (e.g., "Create a photo of me riding a horse").

## Current Tasks

- [ ] Integrate Google Gemini API for personalized image generation
- [ ] Create Gemini Studio UI (prompt input + persona selection + generation)
- [ ] Integrate Veo3 for personalized video generation
